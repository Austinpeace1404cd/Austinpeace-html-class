How to Code HTML on Your Android Phone

You do not need a computer to start building websites! This guide will teach you how to write and test your code directly on your Android device.

Your Coding Guide

Lesson 1

Using Your File Manager

Before writing code, you need a dedicated folder to keep your files organized:

  1. Open your phone's default File Manager app.
  2. Go to your Internal Storage and create a new folder. Name it MyWebProject.
  3. Inside this folder, create a new file and name it index.html.
    Note: If your file manager does not let you change the extension from .txt to .html, you can perform this step inside the TrebEdit app.
Lesson 2

Coding with TrebEdit & HTML Notepad

To type your code, you should download a mobile code editor from the Google Play Store:

  • TrebEdit: A complete web design editor. You can create projects, write tags easily with its auto-completion feature, and run code with a built-in play button.
  • HTML Notepad: A simpler, lightweight app designed for writing quick HTML pages without unnecessary options.

Open either app, select Open File, navigate to your MyWebProject folder, and open your index.html file to begin writing tags.

Lesson 3

Testing Your Work in Google Chrome

Once you write your HTML code and save the file, you can view your creation:

  1. If using TrebEdit, you can tap the built-in Play icon in the editor to see your code run.
  2. To open it in Google Chrome, open your File Manager, find your index.html file, tap it, and select Open With -> Google Chrome.
